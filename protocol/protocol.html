TCP/IP

The foundation of internet communication, responsible for reliable data transmission.

  • TCP (Transmission Control Protocol): Ensures data arrives intact and in order.
  • IP (Internet Protocol): Routes packets across networks.

HTTP & HTTPS

Protocols for accessing web pages and APIs.

  • HTTP (HyperText Transfer Protocol): Standard for web communication.
  • HTTPS: Secure version of HTTP, encrypted using SSL/TLS.

DNS

Translates domain names into IP addresses.

  • Helps users access websites using names instead of numbers.
  • Works with recursive and authoritative name servers.

DHCP

Automatically assigns IP addresses to devices on a network.

  • Reduces manual IP configuration.
  • Uses a lease system to manage addresses dynamically.

FTP & SFTP

Protocols for file transfers between computers.

  • FTP (File Transfer Protocol): Unencrypted file transfer.
  • SFTP (Secure FTP): Secure file transfer over SSH.

ICMP

Used for network diagnostics and error reporting.

  • Foundation of the ping command.
  • Helps identify connectivity issues.
